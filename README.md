# Оптимизация маршрутов

## Описание
Программа для планирования маршрутов между городами с учетом нескольких критериев оптимизации: расстояния, времени и стоимости. Система находит оптимальные маршруты по каждому критерию отдельно, а затем выбирает компромиссный маршрут на основе заданных приоритетов.

## Функциональность
- Поиск оптимальных маршрутов по трем критериям:
  - **ДЛИНА** - минимальное расстояние
  - **ВРЕМЯ** - минимальное время в пути  
  - **СТОИМОСТЬ** - минимальная стоимость
- Выбор компромиссного маршрута на основе приоритетов
- Обработка сложных сетей дорог между городами
- Учет двустороннего движения на всех дорогах
